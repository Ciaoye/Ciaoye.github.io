## 新增 File Manager 应用 + Chat 自动存档

### 新建文件
- `js/apps/fileManager.js` — 文件管理器应用，Win95 风格 UI

  - 显示文件名、类型、大小、日期
  - 下载文件到本地（通过 Blob URL 触发下载）

- `index.html` — 引入 fileManager.js，注册 Files 应用到桌面（icon: image-5.png）
- `js/apps/chat.js` — 新增聊天记录自动存档和恢复

  - 从单回调改为回调数组，支持多个 App 各自注册 close/open/focus 回调
