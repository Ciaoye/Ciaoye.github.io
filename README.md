# ciao os

一个浏览器里的 Win95 风格桌面。聊天、写信、画画、听歌、翻文件、玩扫雷。

**[线上地址](https://ciaoye.github.io/)**

---

## 有什么

| 图标 | 是什么 |
|------|--------|
| Chat | AI 聊天，像发短信一样 |
| Letter | AI 笔友，写长信 |
| Mood Map | 2008 年至今的情绪可视化 |
| Wiki Map | 个人知识图谱 |
| Paint | 像素画板 |
| Files | 文件管理器，存画作和聊天记录 |
| Notepad | 记事本 |
| Browser | 内嵌浏览器 |
| Lofi Radio | 7 个电台，像素风播放器 |
| 2048 | 经典合并游戏 |
| Sweeper | 扫雷 |

还有可拖拽的便利贴、桌面时钟。

---

## 技术

纯静态前端，HTML + CSS + 原生 JS。IndexedDB 做文件系统，Chat/Letter 接 AI 后端。
